# HH.ru Auto Reply

Автоматический скрипт для отклика на вакансии на HH.ru.

## Возможности

- Автоматический клик на кнопку "Откликнуться" для каждой вакансии
- Пропуск уже отправленных откликов
- Обработка модальных окон
- Защита от частых запросов (random delays)
- Генерация отчёта в файл

## Требования

- Python 3.10+
- [Playwright](https://playwright.dev/python/)

## Установка

Создайте виртуальное окружение и установите зависимости:

```bash
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
# или: venv\Scripts\activate  # Windows

pip install playwright
playwright install webkit
```

## Использование

```bash
python hh_auto_reply.py
```

В первый раз запустите скрипт, чтобы установить браузер. Последующие запуски:

```bash
source venv/bin/activate  # Linux/macOS
# или: venv\Scripts\activate  # Windows

python hh_auto_reply.py
```

Процесс работы:

1. Запускается браузер с открытым hh.ru
2. Скрипт ожидает загрузку вакансий (макс. 2 мин)
3. Вы авторизуетесь и настроите фильтры вакансий
4. Нажмите Enter когда готовы
5. Скрипт пройдёт по всем вакансиям и отправит отклики
6. Результат сохраняется в `hh_report.txt`

Браузер запускается в видимом режиме (headful), чтобы вы могли контролировать процесс.

## Настройка

Параметры можно изменить в начале файла:

| Параметр | Описание | По умолчанию |
|----------|-----------|--------------|
| `SLOW_MO` | Замедление действий (мс) | 80 |
| `SELECTOR_RESPONSE` | Селектор кнопки отклика | vacancy-serp__vacancy_response |
| `SLOW_MO` | Задержка между откликами (мс) | 2000-3500 |
