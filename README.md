# HH.ru Auto Reply Script

Автоматический скрипт для отправки откликов на вакансии на hh.ru с помощью Playwright.

## Описание

Скрипт `hh_auto_apply.py` открывает браузер, позволяет вам авторизоваться на hh.ru, настроить фильтры и затем автоматически переходит по страницам результатов вакансий. Он ищет кнопки «Откликнуться», кликает по ним и пытается закрыть всплывающие модальные окна, которые появляются при отправке отклика.

Скрипт также обрабатывает редиректы, всплывающие диалоги и новые вкладки, чтобы снизить вероятность зависания на одной вакансии.

## Основные возможности

- выбор браузера: Chromium или WebKit
- автоматический проход по страницам результатов
- обнаружение и закрытие модальных окон hh.ru
- игнорирование редиректов и возврат на основную страницу
- сохранение отчёта в файл `hh_report.txt`

## Настройки и важные параметры

Файл скрипта: `hh_auto_apply.py`

Ключевые параметры, которые можно изменить прямо в коде:

- `max_pages = 25` — максимальное количество страниц результатов, которые обрабатывает скрипт
- `viewport` и `user_agent` — настройки контекста браузера
- `locale="ru-RU"` и `timezone_id="Europe/Moscow"` — локализация страницы
- селекторы кнопок и модальных окон, определённые в функции `close_response_modal`

## Требования

- Python 3.8+ (рекомендуется Python 3.11+)
- Playwright
- Интернет и рабочий доступ к hh.ru

> На macOS с Apple Silicon (M1/M2/M3/M4) рекомендуется запускать через WebKit, он может работать стабильнее в данном окружении.

## Установка

1. Перейдите в папку проекта:

```bash
cd path/to/your/project
```

2. Создайте виртуальное окружение:

```bash
python3 -m venv venv
```

3. Активируйте виртуальное окружение:

```bash
source venv/bin/activate
```

4. Установите зависимости:

```bash
pip install --upgrade pip
pip install playwright
```

5. Установите браузеры Playwright:

```bash
playwright install chromium webkit
```

> Если нужно установить все доступные движки, можно выполнить `playwright install`.

## Запуск

1. Убедитесь, что виртуальное окружение активировано:

```bash
source venv/bin/activate
```

2. Запустите скрипт:

```bash
python hh_auto_apply.py
```

3. Введите выбор браузера:

- `1` — Chromium
- `2` — WebKit

4. Дождитесь открытия браузера, авторизуйтесь на hh.ru и настройте фильтры поиска.
5. Нажмите Enter в терминале, когда будете готовы начать автоматический обход вакансий.

## Как работает скрипт

- открывает hh.ru
- ждёт, пока пользователь залогинится и настроит фильтры
- проходит по результатам поиска
- кликает кнопки "Откликнуться"
- закрывает модальные диалоги
- возвращается назад, если сработал редирект
- переходит на следующую страницу результатов
- сохраняет отчёт в `hh_report.txt`

## Результаты

После завершения работы скрипт выводит статистику:

- количество обработанных страниц
- успешные отклики
- пропущенные вакансии из-за редиректов
- количество закрытых вкладок/модалок
- количество ошибок

## Примечания

- Скрипт запускается в интерактивном режиме и требует ручного входа на hh.ru перед началом автоматики.
- Использование автоматизации может нарушать правила сайта, применяйте на свой страх и риск.
- Если структура сайта hh.ru изменилась, возможно потребуется обновить селекторы в `hh_auto_apply.py`.

## Файлы проекта

- `hh_auto_apply.py` — основной скрипт
- `hh_report.txt` — автоматически создаётся после завершения работы со скриптом
